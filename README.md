# Cadastro de Usuários - CRUD

Projeto full stack simples de cadastro de usuários, desenvolvido com React no frontend e Node.js com Express no backend.

O projeto aplica os conceitos básicos de CRUD (Create, Read, Update e Delete) e tem como objetivo praticar a comunicação entre frontend e backend utilizando Axios e rotas REST.

---

## Funcionalidades

- Criar usuários
- Listar usuários cadastrados
- Editar dados do usuário
- Excluir usuários
- Atualização da interface sem recarregar a página

---

## Tecnologias Utilizadas

Frontend:
- React
- Axios
- React Router
- CSS Modules


Os dados são armazenados em memória (array) para fins educacionais.
